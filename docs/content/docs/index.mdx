## Features

- **Type-Safe** - Full TypeScript support with strict mode
- **Modular** - Monorepo structure with pnpm workspaces
- **Modern** - ESM-only with latest tooling
- **Fast** - Turbo for caching and parallel builds
- **Complete** - Testing, linting, formatting, and docs included

## Project Structure

```
├── packages/           # Workspace packages
│   └── core/          # Example core package
├── docs/              # Documentation site (Fumadocs)
├── .changeset/        # Changeset configuration
├── .github/           # GitHub Actions workflows
├── turbo.json         # Turbo configuration
└── tsconfig.base.json # Base TypeScript config
```
