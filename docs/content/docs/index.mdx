# Hydra SDK

A type-safe TypeScript SDK for managing [Hydra Head](https://hydra.family/) protocol interactions on Cardano. Built with [Effect](https://effect.website/) for robust error handling, resource management, and composability.

## Features

- **Type-Safe** - Full TypeScript support with strict mode
- **Modular** - Monorepo structure with pnpm workspaces
- **Modern** - ESM-only with latest tooling
- **Fast** - Turbo for caching and parallel builds
- **Complete** - Testing, linting, formatting, and docs included

## Packages

| Package | Description |
|---------|-------------|
| `@no-witness-labs/hydra-sdk` | Core SDK — Head lifecycle, WebSocket transport, protocol types |
| `@no-witness-labs/hydra-sdk-cli` | CLI tool for head lifecycle commands |
| `@no-witness-labs/hydra-devnet` | Local Docker-based Cardano + Hydra development environment |

## Architecture

```
@no-witness-labs/hydra-sdk
├── Head      # HydraHead client — lifecycle commands, event subscriptions
├── Protocol  # Hydra protocol message schemas (Effect Schema)
├── Socket    # WebSocket transport with reconnection
└── Config    # Connection configuration

@no-witness-labs/hydra-devnet
├── Cluster   # Docker-based Cardano + Hydra cluster orchestration
├── Container # Docker container lifecycle management
├── Genesis   # Key generation and genesis config building
└── Health    # Service health check polling
```
