# Getting Started

## Prerequisites

- Node.js 18+
- pnpm 8+
- A running [Hydra node](https://hydra.family/) (or use `@no-witness-labs/hydra-devnet` for local development)

## Installation

### SDK

```bash
pnpm add @no-witness-labs/hydra-sdk
```

For local development with a devnet cluster:

```bash
pnpm add @no-witness-labs/hydra-devnet
```

### CLI

Install the standalone binary (no Node.js required):

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://raw.githubusercontent.com/no-witness-labs/hydra-sdk/main/packages/hydra-sdk-cli/install.sh | sh
```

See [CLI Usage](/docs/cli) for full documentation.

## Quick Example

Connect to a Hydra node and run through the head lifecycle:

```ts
import { Head } from "@no-witness-labs/hydra-sdk";

const head = await Head.create({ url: "ws://localhost:4001" });

// Subscribe to events
const unsubscribe = head.subscribe((event) => {
  console.log(`Event: ${event.tag}`, event.payload);
});

// Initialize a new head
await head.init();
console.log(head.getState()); // "Initializing"

// Close and finalize
await head.close();
await head.fanout();
console.log(head.getState()); // "Final"

unsubscribe();
await head.dispose();
```

## Using the Bracket Pattern

`Head.withHead` ensures the head is always disposed, even if an error occurs:

```ts
import { Head } from "@no-witness-labs/hydra-sdk";

await Head.withHead({ url: "ws://localhost:4001" }, async (head) => {
  await head.init();
  // ... head operations ...
  await head.close();
  await head.fanout();
  // head is automatically disposed on exit
});
```

## Using the Effect API

Every operation is also available as an Effect:

```ts
import { Head } from "@no-witness-labs/hydra-sdk";
import { Effect } from "effect";

const program = Effect.scoped(
  Effect.gen(function* () {
    const head = yield* Head.effect.createScoped({
      url: "ws://localhost:4001",
    });

    yield* head.effect.init();
    yield* head.effect.close();
    yield* head.effect.fanout();
  }),
);

await Effect.runPromise(program);
```

## Local DevNet

Spin up a local Cardano + Hydra cluster with Docker:

```ts
import { Cluster } from "@no-witness-labs/hydra-devnet";
import { Head } from "@no-witness-labs/hydra-sdk";

await Cluster.withCluster(async (cluster) => {
  console.log(`Hydra API: ${cluster.hydraApiUrl}`);

  const head = await Head.create({ url: cluster.hydraApiUrl });
  await head.init();
  // ...
  await head.dispose();
  // cluster is automatically stopped and removed on exit
});
```

## Configuration

### Head Config

```ts
import { Head } from "@no-witness-labs/hydra-sdk";

const head = await Head.create({
  url: "ws://localhost:4001",
  historyOnConnect: true,    // request message history on initial connect
  historyOnReconnect: false, // request history on reconnect
  reconnect: {
    maxRetries: 10,          // default: 10
    initialDelayMs: 500,     // default: 500
    maxDelayMs: 30_000,      // default: 30000
    factor: 1.7,             // exponential backoff factor
    jitter: 0.2,             // jitter factor
  },
});
```

### DevNet Config

```ts
import { Cluster } from "@no-witness-labs/hydra-devnet";

const cluster = Cluster.make({
  clusterName: "my-devnet",
  cardanoNode: {
    image: "ghcr.io/intersectmbo/cardano-node:10.5.3",
    port: 3001,
    networkMagic: 42,
  },
  hydraNode: {
    image: "ghcr.io/cardano-scaling/hydra-node:1.2.0",
    apiPort: 4001,
    contestationPeriod: 60,
  },
});
```
