# API Reference

This section contains API documentation for all packages in the Hydra SDK monorepo.

This reference documents **v2.0.0 (upcoming)** of the SDK, which targets hydra-node 2.3.0. For **v1.0.0** (the current npm `latest`, targeting hydra-node 1.2.0), see the docs at the [v1.0.0 tag](https://github.com/no-witness-labs/hydra-sdk/tree/v1.0.0/docs). The API is versioned under [semantic versioning](https://semver.org); to see what changed between versions, check the [Release Notes](/docs/release-notes) or install a specific version with `npm i @no-witness-labs/hydra-sdk@<version>`.

## Packages

- [@no-witness-labs/hydra-sdk](/docs/modules/hydra-sdk) — Core SDK for Hydra Head protocol management
- [@no-witness-labs/hydra-devnet](/docs/modules/devnet) — Local Docker-based Cardano + Hydra development environment
