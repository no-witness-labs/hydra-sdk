# @template/core

Core utilities and functions for the TypeScript Project Template.

## Functions

### add

Adds two numbers together.

**Signature:**

```ts
declare const add: (a: number, b: number) => number
```

**Parameters:**

- `a` - First number
- `b` - Second number

**Returns:** The sum of a and b

**Example:**

```ts
import { add } from "@template/core"

const result = add(2, 3)
// result = 5
```

---

### subtract

Subtracts the second number from the first.

**Signature:**

```ts
declare const subtract: (a: number, b: number) => number
```

**Parameters:**

- `a` - First number
- `b` - Second number

**Returns:** The difference of a and b

**Example:**

```ts
import { subtract } from "@template/core"

const result = subtract(5, 3)
// result = 2
```
