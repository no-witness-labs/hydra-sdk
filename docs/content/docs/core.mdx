# Core Package

The `@template/core` package provides example utilities to demonstrate the template structure.

## Installation

```bash
pnpm add @template/core
```

## Usage

```ts twoslash
import { add, subtract } from "@template/core"

const sum = add(2, 3)
```

## Examples

### Adding numbers

```ts twoslash
import { add } from "@template/core"

const result = add(2, 3)
```

### Subtracting numbers

```ts twoslash
import { subtract } from "@template/core"

const result = subtract(5, 3)
```
